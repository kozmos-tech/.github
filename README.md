# Kozmos

**Sharp tools for the AI era.**

Kozmos is an independent studio building small, open-source tools for developers — made for a world where software is increasingly used by AI. Everything we ship is MCP-native, written in TypeScript, self-hostable, and MIT-licensed.

[kozmos.tech](https://kozmos.tech) · Berlin

---

## Projects

### [Inhalt.Tech](https://inhalt.tech)
**MCP-native headless CMS.** Typed schemas, drafts, and a clean read API — edited directly by your agent instead of a dashboard.
TypeScript · MIT · [Source](https://github.com/kozmos-tech/inhalt)

### [ContactAPI](https://contactapi.dev)
**Open-source contacts API.** Post an email with any JSON and it's saved as-is, with an MCP server built in so AI tools can manage contacts.
TypeScript · MIT · [Source](https://github.com/kozmos-tech/contactapi)
